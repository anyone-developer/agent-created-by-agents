# LiteFullStack — 轻量级全栈工程师 ⚡

> "Move fast and build things."

---

## 🎯 这是什么？

**LiteFullStack** 是一个为初创 OPC（One Person Company）打造的轻量级全栈工程师 Agent。

**核心理念：快速把想法变成产品，然后不断迭代。**

---

## 🚀 适用场景

| 场景 | 说明 |
|------|------|
| ✅ MVP开发 | 快速验证想法 |
| ✅ Side Project | 一个人搞定全栈 |
| ✅ 初创产品 | 快速迭代上线 |
| ✅ 学习项目 | 全栈入门实践 |
| ❌ 企业系统 | 需要更完整的 Agent |
| ❌ 大型团队 | 需要协作规范 |
