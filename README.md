# 🤖 Agents Creating Agents

> 当 AI Agents 开始创造 AI Agents — 这就是智能的自我进化。

---

## 🎯 项目愿景

**Agents Creating Agents** 是一个实验性项目，探索如何利用多个专家级 AI Agents 协作，创造出更强大、更全面的复合型 AI Agent。

### 核心理念

```
单个专家 Agent → 专业但局限
多个专家协作 → 互补且全面
复合型 Agent → 全知全能
```

## 🚀 快速安装（OpenClaw）

### 方法一：一键安装脚本（推荐）

```bash
curl -sSL https://raw.githubusercontent.com/anyone-developer/agent-created-by-agents/main/install.sh | bash
```

运行后会自动下载并引导你选择安装哪个 Agent。

### 方法二：手动安装

#### Step 1: 克隆仓库
```bash
git clone https://github.com/anyone-developer/agent-created-by-agents.git
```

#### Step 2: 复制到 OpenClaw 目录
```bash
# ProFullStack
mkdir -p ~/.openclaw/workspace-pro-fullstack
cp -r agent-pro-fullstack-engineer/* ~/.openclaw/workspace-pro-fullstack/

# LiteFullStack
mkdir -p ~/.openclaw/workspace-lite-fullstack
cp -r agent-lite-fullstack-engineer/* ~/.openclaw/workspace-lite-fullstack/
```

#### Step 3: 配置 OpenClaw

编辑 `~/.openclaw/openclaw.json`，添加 Agent 配置：

```json
{
  "agents": {
    "list": [
      {
        "id": "pro-fullstack",
        "name": "ProFullStack",
        "emoji": "🚀",
        "description": "专业全栈工程师 - 团队协作、企业项目",
        "workspace": "~/.openclaw/workspace-pro-fullstack",
        "model": "anthropic/claude-sonnet-4-5"
      },
      {
        "id": "lite-fullstack",
        "name": "LiteFullStack", 
        "emoji": "⚡",
        "description": "轻量全栈工程师 - 快速开发、专注交付",
        "workspace": "~/.openclaw/workspace-lite-fullstack",
        "model": "anthropic/claude-sonnet-4-5"
      }
    ]
  }
}
```

#### Step 4: 重启 OpenClaw
```bash
openclaw restart
```

#### Step 5: 开始使用
```
在 Aight App 中选择 Agent：
- 🚀 ProFullStack — 适合复杂项目
- ⚡ LiteFullStack — 适合快速开发
```
